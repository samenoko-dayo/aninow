# AniNow

今期放送中のアニメをまとめてチェックできます！

## About

今期放送中のアニメ情報をまとめてチェックできるwebサイトです。  
現在放送中のアニメ情報と今期のアニメをチェック可能です。

情報のソースは[Annict](https://annict.com)です。

## Stack

- Next.js v16
- React v16
- Cloudflare Pages

## 情報の更新

毎日15:00(UTC),日本時間だと深夜0時に情報の更新を行います。  
ページのトップに更新した時間を表示しています。  
Actionsの仕様により若干ズレる場合があります。
